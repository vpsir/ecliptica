# Ecliptica 🌑

Ecliptica is a high-precision, frontend-only eclipse computation engine built with modern JavaScript and VSOP87-based celestial mechanics.

It calculates lunar and solar eclipse circumstances — including precise local contact times — directly in the browser, with no external APIs.

Powered by:
- VSOP87 planetary theory
- High-accuracy lunar periodic terms
- ΔT modeling
- Nutation & aberration corrections
- Topocentric coordinate transformations
- Numerical contact time solving

Ecliptica brings astronomical-grade 🚀 eclipse prediction ⌚ to the open web. 🌐
